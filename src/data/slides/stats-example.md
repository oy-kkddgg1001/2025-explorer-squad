# 통계 슬라이드 사용법

## 📋 YAML Front Matter 설정

- `type: stats` - 통계 슬라이드 타입으로 설정
- `stats` 배열에 표시할 통계 데이터 나열
  - `value`: 표시할 값 (숫자나 텍스트)
  - `label`: 값에 대한 설명

## 🎨 레이아웃 옵션

- `layout: horizontal` - 가로 배치 (기본값)
- `layout: vertical` - 세로 배치
- `layout: grid` - 격자 배치

## 💡 활용 팁

- 임팩트가 큰 숫자를 강조하세요
- 간결하고 명확한 라벨을 사용하세요
- 4-6개 정도의 통계가 적당합니다

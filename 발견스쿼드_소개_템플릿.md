# 1. 2025년 발견스쿼드 소개

> **발견스쿼드**

**[스쿼드 포지셔닝]**

- "UGC/크리에이터 콘텐츠 생태계를 구축-디벨롭하는 스쿼드"
- 담당 영역: 리뷰/셔터/발견/어필리에이트 등 커뮤니티 기반 영역 전담

**[중장기 성장축 목표]**

1. 어필리에이트 플랫폼 확장
2. 발견탭 기반 콘텐츠 광고 피드 BM 확장
3. 콘텐츠 중심 유저-크리에이터-파트너 연결 플랫폼 확장

**[브랜드 플랫폼 비전]**

- "커뮤니티 서비스를 넘어 협력사의 성장을 돕는 브랜드 솔루션으로 진화"
- 브랜드사와 일반 사용자/크리에이터 간 커뮤니케이션 플랫폼 구축

**[어필리에이트 목표]**

- 디지털 미디어 대응력 보완
- 플랫폼사 알고리즘 활용한 노출/전환 강화
- 브랜드 지위 및 영향력 제고

## 1. 스쿼드의 목표

유저에게 재미와 구매 의사결정에 도움을 주고, 올리브영 앱 체류 시간을 늘리는 콘텐츠 경험을 만듭니다. 정답과 성공 방정식이 정해져 있지 않은 도전적 영역에서 매번 색다른 시각으로 문제를 해결합니다.

- **콘텐츠 생태계 구축**: UGC/크리에이터 콘텐츠 기반 커뮤니티 활성화
  - 숏폼, 리뷰, 매거진 등 다양한 콘텐츠 포맷 지원
  - 크리에이터-유저-브랜드 연결 플랫폼 강화
- **비즈니스 모델 확장**: 어필리에이트 플랫폼을 통한 새로운 수익 창출
  - 크리에이터 커머스 생태계 구축
  - 브랜드 솔루션 제공
- **기술 고도화**: 모던 아키텍처 전환 및 개발 효율성 향상
  - Web Component 기반 컴포넌트 표준화
  - API 공통화 및 성능 최적화

**[프로젝트 개요]**

**[주요 목표]**
- 숏폼 업로드 기능 개발

- 셔터 메인 개편

### 어필리에이트 프로젝트

**[프로젝트 개요]**
> 크리에이터 커머스 플랫폼을 구축하여 새로운 비즈니스 모델을 창출하는 프로젝트입니다.

**[주요 목표]**
- Phase 1: 기본 어필리에이트 시스템 구축
- Phase 2: 정산금 지급 시스템 완성
- 크리에이터-브랜드 연결 플랫폼 구현

**[핵심 성과]**
- 5일간 총 매출 5,133만원 달성
- Phase 1 배포 8/21 완료 (89/93 티켓, 95%)
- Phase 2 배포 9/18 완료

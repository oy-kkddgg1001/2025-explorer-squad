## 📝 콘텐츠 슬라이드 작성 가이드

### YAML Front Matter 설정

- `type: content` - 콘텐츠 슬라이드 타입
- `title` - 슬라이드 제목
- `layout` - 레이아웃 옵션:
  - `center` - 중앙 정렬 (기본값)
  - `left` - 좌측 정렬
  - `two-column` - 2단 레이아웃

### 마크다운 문법 지원

- `# 제목` - 대제목
- `## 소제목` - 중제목
- `### 항목 제목` - 소제목
- `- 목록 항목` - 불릿 포인트
- `**굵은 글씨**` - 볼드체
- `---` - 구분선
